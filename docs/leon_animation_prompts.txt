----------------------------------------------------------------
【手順】
----------------------------------------------------------------

1. ChatGPT に確定済みのレオン画像（レオン.png）を貼り付ける
2. 最初に下の「セッション開始文」を送る
3. あとは各アニメのプロンプトをそのまま送るだけ
4. 同じチャットセッションを維持することで一貫性が保たれる


----------------------------------------------------------------
【セッション開始文】※画像と一緒に最初に送る
----------------------------------------------------------------

This is the reference character design for Leon.
Please use this exact visual style, outfit, colors, and proportions
for all sprite animations I will request in this session.
Art style: pixel art, clean sharp outlines, no anti-aliasing,
transparent background, same style as KOF / SNK fighting games.
Camera angle: 2D side-scrolling fighting game perspective (like KOF or Street Fighter).
The character always faces right. Do NOT use a front-facing or 3/4 perspective.
Each frame size: 64x128 pixels.
Confirm you have memorized this character before I send animation requests.

【うまくいかないときの追加文】
----------------------------------------------------------------

カメラ視点がおかしい場合（正面・斜めになる）:
"Side view only. This is a 2D fighting game like KOF or Street Fighter.
The character must be viewed from the side, NOT front-facing, NOT 3/4 angle,
NOT walking toward the camera."

横並びにならない場合:
"Please place all frames in a single horizontal row, left to right, no vertical stacking."

背景が白い場合:
"The background must be fully transparent (no white background)."

キャラのデザインがブレた場合:
"Please refer back to the original Leon reference image I sent at the start of this session."
